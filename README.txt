Xampp Install
-------------
Disable Antivirus
Install Xampp as administrator

Run the code
------------
Disable Antivirus
Copy index.php and chat_server.php to C:\xampp\htdocs
From an Xampp Shell
   php -q C:\xampp\htdocs\chat_server.php
From Chrome visit
   localhost
